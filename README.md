### ゲームの総論
```
１．全てはダイスの元に公平である。
ダイスは、aDb+c方式とする。aはレベルを指し、bはレベルに対する強さを指す。cは最低値を保証する質である。
最大レベルもダイスによる。例えば、9D20、は魔物の最大生命力を示す。
魔法や武器の威力もabcを基準にする事で、バランスの不平等を無くす。
例えば、魔物の最大レベルが９であれば、魔法や武器もそれに即したものにする。
しかし、６人パーティーを考えれば、魔法や武器の最大レベルは７でよい。
２．無駄な魔物をなくす。全ての魔物にエピソードを加えたものをゲームとする。
クエストに関しても、魔物をできるだけ使い、魔物の個性に合わせたサブクエストとする。
３．道は無数に提示するが、選択を選べる程度で、指針や方針は決める。
システムの解放も多くはクエストによるものとする。メモとして、画面に幾つかクエストを貼っておく。
複数の層にクエストを渡らせない。ひとつのクエストはひとつの層で完結させる。メインシナリオはやや複雑でも良いが――。
４．回復は絞る。蘇生も唯一とするのがよい。
５．魔物は話す。魔物は硬派な生活をさせる。

```
