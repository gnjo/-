### ゲームの総論
```
１．全てはダイスの元に公平である。
ダイスは、aDb+c方式とする。aはレベルを指し、bはレベルに対する強さを指す。cは最低値を保証する質である。
最大レベルもダイスによる。例えば、9D20、は魔物の最大生命力を示す。
魔法や武器の威力もabcを基準にする事で、バランスの不平等を無くす。
例えば、魔物の最大レベルが９であれば、魔法や武器もそれに即したものにする。
しかし、６人パーティーを考えれば、魔法や武器の最大レベルは７でよい。
２．無駄な魔物をなくす。全ての魔物にエピソードを加えたものをゲームとする。
クエストに関しても、魔物をできるだけ使い、魔物の個性に合わせたサブクエストとする。
３．道は無数に提示するが、選択を選べる程度で、指針や方針は決める。
システムの解放も多くはクエストによるものとする。メモとして、画面に幾つかクエストを貼っておく。
複数の層にクエストを渡らせない。ひとつのクエストはひとつの層で完結させる。メインシナリオはやや複雑でも良いが――。
４．回復は絞る。蘇生も唯一とするのがよい。
魔法の重要性を示すためにも、消耗品のアイテムは少なくしたほうが良い。
回復を絞る場合、ターンによって回復する武具、ターン回復の武具を加えるとバランスが取れる。
消耗品を装備制とし、確率で消失するようになる。例えば、回復薬を装備し、一度使うが確率に当選すれば、回復薬は消耗しない。
５．魔物は話す。魔物は硬派な生活をさせる。
魔物は人間のファインダーであるので、人間臭いほうがよい。
６．魔法は群戦に強く、武器は単体に強い。
差別化させる。武器に範囲攻撃は厳禁。
７．クエストは濃く作る。水増しはしない。
８．転生や引退は極大にする。
例えば、引退した者の魔法を一つだけ、魔法具化できるようにする。伝説の武具システムなど。
引退可能なレベルの最低は１３。
＞熟練の冒険者の一人であるファテマが引退しました。その者が残した伝説の武具とは何ですか？
＞ファテマのマント
９．拠点は小さく、店は倉庫。
魔物を倒す事が大半のゲームにおいて、魔物から産出される武具の収集はゲームの目的の一つである事が多い。
拠点に長く滞在する事を避けるために、拠点は小さくする。
宿屋：レベルを上げる機能を持つ。
酒場：クエストを受ける機能を持つ。
商店：倉庫と売買の機能を持つ。
寺院：蘇生や浄化の機能を持つ。寄付による経験値上昇機能を持つ。
王宮：メインクエストの報告の機能を持つ。
迷宮：大半はここで過ごす。
１０．ソードユーザ、マジックユーザ、マスターの区分けを持つ。
レベル１３までは、前衛向けのソードユーザと後衛向けのマジックユーザとする。
レベル１３以降は、ソードマスターとマジックマスターとなる。
ソードマスターは、攻撃回数が二回増える。
マジックマスターは、五割の確率で無消費魔法が発生する。
１１．フラグは全てアイテムで持つ。
重要アイテムの有無のみで、シナリオが進行できると良い。
```
```
これらを踏まえて、七章仕立てのシナリオを考える。
七章であるので、レベルは７とする。
9D20 //最も強い魔物
7D20 //一般の魔物の強さの最大値
7D12 //魔法の威力の最大値
7D8  //武器の威力の最大値。両手が可能であるため。
7 //迷宮の層の数
7 //大物の魔物の数
7 //そこそこ強い魔物の数
7 //各層のNPCの数、魔物も話すので全て人間である必要はない。
50 //NPCの総数であり、クエストの数。
50 //7x7~50//各層に新種の魔物が七体居る。最低でも５０は必要。
```
